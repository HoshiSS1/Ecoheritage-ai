# Thư mục Hình ảnh Bản đồ Di sản

Bạn hãy chép các hình ảnh trực quan mà bạn muốn dùng cho Bản đồ Di sản (Khu bảo tồn, Tiệm di sản, Vùng dược liệu...) vào thư mục này nhé.

Sau khi thêm ảnh vào đây, hãy cho tôi biết tên các file ảnh đó (ví dụ: `rung-ba-na.jpg`, `vuon-ngu-hanh-son.png`, v.v.), tôi sẽ cập nhật lại file `HeritageMap.tsx` để hiển thị chính xác các ảnh của bạn lên bản đồ!
